# Glossary

Shared terms across GUB documentation and Skull King rules.

| Term | Definition |
|------|------------|
| **Artifact** | Optional expansion or house-rule toggle (Kraken, Loot, Pirate Abilities, etc.). See [artifacts matrix](../games/skull-king/reference/artifacts-matrix.md). |
| **Bid** | Number of tricks a player commits to win in a round (0 through hand size). |
| **Black suit** | Jolly Roger trump suit; outranks green, yellow, and purple. |
| **Capture bonus** | Points for taking a character card in a trick (Mermaid→Pirate, Pirate→Skull King, Mermaid→Skull King). Requires correct bid. |
| **Deal** | Cards dealt for one round; hand size usually equals round number (with caps at high player counts). |
| **Escape** | Special card that always loses the trick (unless all players play Escape). |
| **Follow suit** | Must play a card of the led standard suit when you hold one. |
| **GUB** | Game hub — this application. |
| **Hand size** | Number of cards dealt to each player in the current round. |
| **Lead** | First card played in a trick; sets follow-suit rules. |
| **Loot alliance** | Bonus when Loot card links two players who both make their bids (+20 each). |
| **Match** | A full Play-mode session with join code and server state. |
| **Mermaid** | Special card; beats Skull King and suited cards; loses to Pirates. |
| **Pirate** | Special card; beats suited cards; ties broken by first played. |
| **Preset** | Bundled artifact toggles: beginner, intermediate, expert. |
| **Round** | One deal + bid + tricks + score in a voyage. |
| **Round schema** | Sequence of hand sizes per round (e.g. `1,2,3,…,10`). |
| **Skull King** | Special card; beats Pirates; loses to Mermaids. |
| **Special card** | Non-suited cards (Escape, Pirates, characters, advanced). May be played even when you could follow suit. |
| **Suited card** | Numbered card in green, yellow, purple, or black (1–14). |
| **Trick** | One card played by each player; highest per rules wins. |
| **Tigress** | Special card played as either Pirate or Escape (declare when played). |
| **Voyage** | Full game from first round through final scoring (Calculator term). |
| **Zero bid** | Bidding 0 tricks; special scoring (+10× hand size if exact, −10× if miss). |
